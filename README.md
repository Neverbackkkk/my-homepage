# 我的个人主页项目

本项目用于算法基础课程大作业，使用 AI Agent 辅助搭建个人主页并发布第一篇技术博客。

## 文件说明

- `index.html`: 网站首页。
- `blog/first-blog.html`: 第一篇技术博客。
- `style.css`: 页面样式。
- `script.js`: 简单交互。
- `AGENTS.md`: AI Agent 协作规范。

## 本地查看

直接用浏览器打开 `index.html` 即可预览网站。

## 部署建议

可以将本项目上传到 GitHub 仓库，然后在 GitHub Pages 中选择 `main` 分支和根目录发布。
