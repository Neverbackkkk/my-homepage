# AI Agent 如何帮助我从零搭建个人主页

## 博客主题

AI Agent、学习方法与个人主页搭建。

## 内容草稿

这次课程大作业要求完成一个个人主页，并在主页中发布第一篇技术博客。对零基础学习者来说，网页开发一开始会显得很陌生：既要理解文件结构，又要知道 HTML、CSS 和部署之间的关系。

我的做法是先阅读作业要求，再把任务拆成几个小步骤：确认网页必须包含个人介绍、博客入口和可访问链接；设计首页和博客页结构；让 AI Agent 辅助生成代码；最后由我自己检查链接、内容和页面效果。

通过这个过程，我发现 AI Agent 可以快速生成初稿，也可以帮助整理检查清单，但它不能替代个人判断。主页上的个人信息、博客中的学习感受，以及最终是否符合课程要求，都需要由我自己确认和修改。

提交前需要继续把这篇草稿改成自己的真实经历，并补充部署过程和截图。
