# AGENTS.md

## Project Goal
完成算法基础课程大作业：搭建一个可以部署到 GitHub Pages 的个人主页，并发布第一篇技术博客。

## Important Files
- `index.html`: 个人主页入口，部署时应作为网站首页。
- `blog/first-blog.html`: 第一篇技术博客页面。
- `style.css`: 全站样式。
- `script.js`: 导航交互和年份显示。
- `blog/first-blog.md`: 博客草稿，方便继续修改文字。
- `docs/ai/*`: AI Agent 工作流记录，可用于写作业报告。

## Working Rules
- 保持页面简洁、真实、可读。
- 所有链接修改后都要检查能否正常跳转。
- 个人信息和博客观点需要由学生本人确认，AI 只负责辅助生成、整理和润色。
- 不引入复杂框架，优先使用 HTML、CSS 和少量 JavaScript。

## Verification
1. 首页能否正常打开。
2. 首页能否跳转到博客页。
3. 博客页能否返回首页。
4. 窗口缩小时页面是否仍然可读。
5. 提交前是否已替换姓名、邮箱、GitHub 等个人信息。
